API - Overview

We'll be defining the API standards according to OpenAPI Standards. We'll be consistently pushing YAML files that'll translate into the same. Example of OpenAPI standards

Reference link : https://spec.openapis.org/oas/v3.1.0

It works by translating the YAML to JSON and gives us a standard usable by Swagger.
