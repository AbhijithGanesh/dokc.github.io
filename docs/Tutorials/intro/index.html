Installing this website

This website is built using Docusaurus 2, a modern static website generator.

Source to this website

Clone the repository using

$ git clone https://github.com/dokc/dokc.github.io

Installation‚Äã

$ yarn

Local Development‚Äã

$ yarn start

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

Build‚Äã

$ yarn build

This command generates static content into the build directory and can be served using any static contents hosting service.

Deployment‚Äã

Using SSH:

$ USE_SSH=true yarn deploy

Not using SSH:

$ GIT_USER=<Your GitHub username> yarn deploy

If you are using GitHub pages for hosting, this command is a convenient way to build the website and push to the gh-pages branch.‚Äã

Getting Started‚Äã

Get started by creating a new site.

Or try Docusaurus immediately with docusaurus.new.

What you'll need‚Äã

  • Node.js version 14 or above:
    • When installing Node.js, you are recommended to check all checkboxes related to dependencies.

Generate a new site‚Äã

Generate a new Docusaurus site using the classic template.

The classic template will automatically be added to your project after you run the command:

npm init docusaurus@latest my-website classic

You can type this command into Command Prompt, Powershell, Terminal, or any other integrated terminal of your code editor.

The command also installs all necessary dependencies you need to run Docusaurus.

Start your site‚Äã

Run the development server:

cd my-website
npm run start

The cd command changes the directory you're working with. In order to work with your newly created Docusaurus site, you'll need to navigate the terminal there.

The npm run start command builds your website locally and serves it through a development server, ready for you to view at http://localhost:3000/.

Open docs/intro.md (this page) and edit some lines: the site reloads automatically and displays your changes.
